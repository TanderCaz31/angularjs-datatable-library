Lista di tutte le query effettuate in questo progetto

Per imparare Bootstrap e allo scopo di documentazione, qui trovi una lista delle query effettuate durante la creazione del database in MySql di questo progetto.



Creazione iniziale delle tabelle

Tabella 'nominativi'

CREATE TABLE nominativi (
id INTEGER AUTO_INCREMENT PRIMARY KEY,
nome VARCHAR NOT NULL,
cognome VARCHAR,
data_nascita DATE,
id_città INTEGER,
email VARCHAR
)

Tabella 'regioni'

CREATE TABLE regioni (
id INTEGER AUTO_INCREMENT PRIMARY KEY,
nome VARCHAR(50)
)

Tabella 'città'

CREATE TABLE città (
id INTEGER AUTO_INCREMENT PRIMARY KEY,
nome VARCHAR(50),
id_regione INTEGER,
FOREIGN KEY (id_regione) REFERENCES regioni(id)
)

Modifica a 'nominativi' per correggere il vincolo

ALTER TABLE nominativi
ADD CONSTRAINT FOREIGN KEY (id_città) REFERENCES città(id)
